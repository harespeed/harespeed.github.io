# 个人主页

这是我的GitHub Pages个人主页，使用HTML和CSS构建。

## 如何部署

1. 在GitHub上创建一个新仓库，命名为 `username.github.io`（其中username是你的GitHub用户名）
2. 将本地仓库推送到GitHub：
   ```bash
   git remote add origin https://github.com/username/username.github.io.git
   git add .
   git commit -m "Initial commit"
   git push -u origin master
   ```
3. 打开浏览器访问 `https://username.github.io` 查看你的个人主页

## 自定义内容

编辑 `index.html` 文件来修改个人主页的内容，包括：
- 个人信息
- 技能列表
- 项目展示
- 社交链接

## 技术栈

- HTML5
- CSS3
